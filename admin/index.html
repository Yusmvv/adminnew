API Ключ для лендингов

Настройки GitHub Gist

Этот ID нужно добавить в код лендингов для загрузки данных

Инструкция для настройки:

1. Создайте GitHub Personal Access Token:

  • Зайдите в GitHub → Settings → Developer settings
  • Personal access tokens → Tokens (classic)
  • Создайте токен с разрешением gist
  • Скопируйте токен (он покажется только один раз!)

2. Создайте Gist:

  • Зайдите на gist.github.com
  • Создайте приватный Gist с файлом offers.json
  • Вставьте {} (пустой JSON) в содержимое
  • Скопируйте ID из URL (например: ...gist.github.com/вашлогин/a1b2c3d4e5f6)

3. Настройте токен в коде:

// В файле админ-панели, строка 710
GITHUB_TOKEN: 'ВАШ_ТОКЕН_ЗДЕСЬ' // Замените на ваш GitHub токен
Важно:
GitHub токен нужен только для админ-панели. Лендинги загружают данные без токена, только по Gist ID.
